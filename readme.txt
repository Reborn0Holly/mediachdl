Загрузчик медиафайлов 2ch/arhivach/4chan

Версия 0.0.3

Графическое приложение для загрузки медиафайлов (изображений и видео) из тредов на сайтах 2ch, arhivach, 4chan. Поддерживает многопоточную загрузку, выбор типов файлов и удобный интерфейс с прогресс-баром и логом.
Возможности

	Загрузка изображений (PNG, JPG, JPEG, WEBP) и видео (WEBM, MP4) из тредов
	Выбор типа загружаемых медиафайлов: все изображения, все видео, все медиа или конкретное расширение.
	Многопоточная загрузка с настраиваемым количеством потоков (рекомендуется не более 3 для стабильности).
	Пропуск существующих файлов или создание копий с уникальными именами.
	Проверка URL перед загрузкой с выводом количества найденных медиафайлов.
	Графический интерфейс с прогресс-баром, логом и статус-баром.
	Сохранение лога загрузки в текстовый файл.
	
	
Требования:

	Python 3.14+
	
	
Установка:

	Скачайте zip архив из вкладки release
	
	Запустите приложение:
	launch.bat
	

Установка:

	Скачайте exe
	
	Запустите приложение

Использование:

    Запустите программу.
    Введите URL треда с в поле ввода
    (Опционально) Нажмите "URL из буфера", чтобы вставить ссылку из буфера обмена.
    Нажмите "Проверить", чтобы убедиться, что в треде есть медиафайлы.
    Выберите тип медиафайлов для загрузки (например, "Все видео" или "PNG").
    Укажите папку для сохранения файлов (по умолчанию — папка Downloads в директории скрипта).
    (Опционально) Настройте параметры: пропуск существующих файлов, количество потоков.
    Нажмите "Начать загрузку" и следите за прогрессом в интерфейсе.
    После завершения можно открыть папку с файлами или сохранить лог.
